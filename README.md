# Face Attendance System

AI-based Face Attendance System using Python, OpenCV, and Face Recognition.

## Features

* Face Detection
* Face Recognition
* Automatic Attendance Marking
* CSV Attendance Storage
* Auto Camera Close After Detection
* Old Attendance Removed After 1 Day

## Technologies Used

* Python
* OpenCV
* Face Recognition
* NumPy

## Installation

pip install opencv-python face-recognition numpy pandas

## Run

python attendance.py
